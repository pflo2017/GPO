Get Started with GPO: Your AI Co-Pilot in Minutes

Everything you need to start using GPO tomorrow in your ecosystem

1. Onboarding Your Team

  1. Invite team members via email invitation
  2. Set up role-based permissions (Admin, PM, Viewer)
  3. Configure your organization's security settings
  4. Import existing linguist database (CSV/API)

2. Creating Your First AI-Powered Project

  1. Click "New Project" and fill in basic details
  2. Upload source documents (DOCX, PDF, TXT)
  3. Select language pair and content type
  4. Set deadline and assign linguist
  5. GPO automatically analyzes and provides risk insights

3. Understanding AI Risk Insights

Risk Status: Low/Medium/High/Critical - based on deadline, progress, and content complexity

Risk Reason: Specific factors contributing to the risk assessment

Recommendations: Actionable steps to mitigate risks and optimize outcomes

Dashboard View: Real-time overview of all project risks and statuses

4. Integrating GPO with Your TMS/ERP (API Overview)

Authentication: Use API keys for secure access

Project Creation: POST to /api/projects to create projects programmatically

Risk Updates: GET /api/projects/{id}/risks for real-time risk status

Webhooks: Receive notifications when risk levels change

Documentation: Complete API docs available in your dashboard
